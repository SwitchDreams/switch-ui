## Avatar:

### Uso Básico:

#### O componente Avatar é usado para exibir avatares de usuário. Aqui está como você pode usá-lo:

<Canvas />

<Controls />

#### O componente avatar por ser algo que é utilizado em diversas telas tem a capacidade de ser alterado utilizando o plugin da switch ui:

#### plugins:

```javascript
plugins: [switchUiPlugin, generated({ prefix: "ui" })],
```

#### Configurado desta forma:

 ```javascript
 avatar: {
          primary: {
            bg: colors.primary["25"],
            text: colors.secondary["400"],
          },
          gray: {
            bg: colors.primary["100"],
            text: colors.gray["600"],
          },
        },
```
