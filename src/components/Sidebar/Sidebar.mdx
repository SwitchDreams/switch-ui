## Cada parte da sidebar possui Props únicas:

#### sideBarMobileColor: essa props é passada para a sidebar no objetivo de mudar a cor do header do mobile.
#### mobileLogoClassName: use essa prop para ajustar o tamanho/posição da logo exibida no header mobile sem impactar os projetos que mantêm o padrão.

### Sidebar Item:

#### Label: É uma string passada para ser o nome exibido.
#### href: Link para o botão.
#### Icon: Ícone que será utilizado na frente do botão.
#### Active: ao passar esse booleano para o componente ele ficara com a cor do hover quando estiver ativo na página.

### Sidebar Drop:

#### Label: É uma string passada para ser o nome exibido.
#### Options: Array de opções contendo um objeto href e label para cada opção desejada no Dropdown.
#### Icon: Ícone que será utilizado na frente do botão.

### Sidebar.Logo:

#### href: Link para o botão.
#### Logo: string para a logo a ser utilizada no modo desktop.

### Sidebar.User

#### title: é o testo que vira ao lado do avatar do usuário.
#### subTitle: é o testo que vira abaixo do title podendo ser qualquer testo.
