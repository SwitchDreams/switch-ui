## TextFieldMask:

<br />

### Uso Básico:

#### O componente `TextFieldMask` oferece a seguinte funcionalidade:
- Renderiza um campo de entrada de texto com suporte a ícones nos lados esquerdo e direito.
- Permite definir um rótulo, dica de entrada e texto de suporte.
- Pode ser estilizado com diferentes tamanhos e estados (erro, desabilitado).

### Diferencial:

- Capacidade de receber mascaras de input com um regex.
- Props maskChar essa props server para passar um char caso o usuário nao preecha a string por completo.
