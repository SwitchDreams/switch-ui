## TextField:

<br />

### Uso Básico:

#### O componente `TextField` oferece a seguinte funcionalidade:
- Renderiza um campo de entrada de texto com suporte a ícones nos lados esquerdo e direito.
- Permite definir um rótulo, dica de entrada e texto de suporte.
- Pode ser estilizado com diferentes tamanhos e estados (erro, desabilitado).


<Canvas />

### Exemplo em JSX:

Aqui está um exemplo de como você pode usar o componente TextField em JSX:

```jsx
import { XMarkIcon } from "@heroicons/react/24/solid";
import { TextField } from "./TextField";

const ExampleTextField = () => {
  return (
    <div>
      <TextField
        label="Nome"
        placeholder="Digite seu nome"
        name="name"
        leftIcon={<XMarkIcon />}
        rightIcon={<XMarkIcon />}
        error={false}
        errorMsg=""
        className="w-full"
      />
    </div>
  );
}

export default ExampleTextField;

```

### Configurações adicionais para projetos:

